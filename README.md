# POE Ninja 中文化插件

> 一個為 [poe.ninja](https://poe.ninja) 提供繁體中文翻譯的 Chrome / Edge 瀏覽器插件

## 功能特色

- **物品名稱**：裝備、通貨、地圖、寶石等，全數對照官方繁體中文
- **主頁雙語顯示**：道具名稱以中文為主、英文小字輔助
- **語言切換**：可隨時切換英文 / 繁體中文

## 字庫來源

翻譯資料來自 **[SnosMe/awakened-poe-trade](https://github.com/SnosMe/awakened-poe-trade)**

## 安裝方式

### 下載 Release

1. 前往 [Releases](../../releases) 下載最新版 `.zip`
2. 開啟瀏覽器的擴充功能頁面
   - Chrome：`chrome://extensions/`
   - Edge：`edge://extensions/`
3. 開啟右上角 **「開發人員模式」**
4. Chrome : 點擊 「載入未封裝項目」，把剛的檔案解壓縮，然後選擇 src 資料夾
   Edge : 載入壓縮檔
5. 前往 [poe.ninja](https://poe.ninja) 並點擊插件圖示切換至繁體中文

## 注意事項

- 本插件僅支援 **Path of Exile 一代（POE1）**
- POE2 不在支援範圍內
- 簡體中文功能預留按鈕，尚未實作
